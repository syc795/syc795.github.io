Hugo博客搭建

Jul 6, 2020 · 1 min read

列举Hugo安装博客的步骤命令（Mac）

1、安装Hugo

#安装Hugo
brew update
brew install hugo
hugo version

2、新建博客并启动

#新建
hugo new site MyBlog
#下载主题并配置
cd themes
git clone https://github.com/alexandrevicenzi/soho.git
vim config.toml
#新建博客文件
hugo new post/blog.md
#编写博客
vim content/post/blog.md
#启动（本地）
hugo server -t soho --buildDrafts

3、上传到Github

#上传生成后的文件到github
cd public
#初始化仓库并提交
git init
git add *
git commmit -m "hugo"
#设置远端仓库地址
git remote add origin https://github.com/syc795/syc795.github.io.git
#push到仓库
git push -u origin master

4、新增、更新博客

#新增博客
hugo new post/blog.md
#编写博客
vim content/post/blog.md
#新建、修改博客后重新生成静态文件
cd public
hugo --buildDrafts
#提交文件
git add *
git push
