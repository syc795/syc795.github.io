Nginx命令

Jul 18, 2020 · 1 min read

Nginx启动

nginx

然后在浏览器中输入：http://localhost:8080/ 查看 Nginx 首页。

Nginx操作

  1. nginx -s reload：重新加载配置

  2. nginx -s reopen：重启

  3. nginx -s stop：停止

  4. nginx -s quit：退出

    其中 Nginx 配置文件路径在 /usr/local/etc/nginx/nginx.conf
